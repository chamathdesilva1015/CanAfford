# Security Policy

## Supported Versions

Currently, only the latest version of CanAfford is supported.

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability within this project, please open an issue or contact the maintainers directly. We will address all security vulnerabilities as quickly as possible.
